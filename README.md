# Racing Dodgio 🚗

A top-down racing and dodging game built in **Godot 4.6**. Drive your car, collect cookies for speed boosts, and dodge the Jeeps!

## How to Play

1. Open the project in [Godot 4.6](https://godotengine.org/)
2. Press **F5** to run the game

### Controls

| Key | Action |
|---|---|
| Arrow Up | Accelerate |
| Arrow Down | Brake / Reverse |
| Arrow Left | Turn left |
| Arrow Right | Turn right |

## Gameplay

- **Collect cookies** to get a 5-second speed boost — and trigger the Griddy dance 🕺
- **Dodge Jeeps** — hitting one stops your car and stuns you for 2 seconds
- Build up speed and see how long you can last

## Built With

- [Godot 4.6](https://godotengine.org/) — GDScript
- Jolt Physics engine
- D3D12 rendering (Windows)
